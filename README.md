## Реальность 2026 года

Открыть интернет-магазин можно за вечер. Tilda, Shopify, WooCommerce — выбрал шаблон, загрузил товар, подключил оплату. Технически всё просто.

Закрыть — ещё проще. 90% магазинов не переживают первый год.

Проблема не в технологиях. Проблема в том, что люди создают «ещё один магазин», а не причину у себя покупать.

```
Количество интернет-магазинов в рунете:

2020  ████████████████░░░░░░░░  ~100 000
2022  ██████████████████████░░  ~180 000
2024  ████████████████████████  ~250 000+
2026  ████████████████████████  ~300 000+

Из них выживают через год:       ~10%
Из них прибыльны через 2 года:   ~3–5%
```

## Три типа магазинов

### 💀 Мёртвые

```
Признаки:
- Сайт на бесплатном шаблоне
- 3–10 товаров
- Описание: «отличное качество, рекомендуем»
- Фото с AliExpress в разрешении 200×200
- Последний заказ: ноябрь прошлого года
- Владелец удивляется почему нет продаж
```

Таких — большинство. Человек посмотрел видео «как заработать миллион на дропшиппинге», сделал сайт за выходные и ждёт денег. Деньги не приходят.

### 🧟 Зомби

```
Выручка:       500 000 ₽/мес
Расходы:       520 000 ₽/мес
─────────────────────────────
Прибыль:       -20 000 ₽/мес

Трафик есть. Заказы есть. Прибыли нет.
```

Самый опасный тип. Внешне всё выглядит нормально — сайт работает, заказы идут, деньги крутятся. Но если посчитать юнит-экономику, выясняется: каждая продажа приносит убыток. Чем больше продаёшь — тем больше теряешь.

### 🟢 Живые

Их мало. Как пример живая онион площадка [blacksprurtblsp.com](https://blacksprurtblsp.com) Но они делают вещи иначе:

| Что делают мёртвые | Что делают живые |
|---|---|
| Копируют фото с поставщика | Делают свои фото как для журнала |
| Описание: «хорошее качество» | Описание отвечает на вопросы до того как их задали |
| Доставка «уточняйте» | Чёткие сроки, цена, трекинг |
| Поддержка отвечает через сутки | Чат с ответом за минуту |
| Продают всё для всех | Узкая ниша, глубокая экспертиза |
| Считают только выручку | Знают юнит-экономику до копейки |
| Игнорируют старых клиентов | Строят повторные продажи |

## Почему магазины умирают

### 1. Нет ответа на главный вопрос

> «Почему покупать здесь, а не на маркетплейсе?»

Покупатель может найти тот же товар на Ozon с бесплатной доставкой и возвратом. Если ваш магазин не предлагает ничего сверх этого — вы проиграли до старта.

Что может быть этим «сверх»:

```
- Уникальный ассортимент (нет на маркетплейсах)
- Экспертная консультация
- Кастомизация товара
- Сильный бренд и история
- Комьюнити вокруг продукта
- Сервис, который маркетплейс не даст
```

### 2. Плохое первое впечатление

```
Время на решение «остаться или уйти»:    3 секунды
Стоимость привлечения посетителя:         50–300 ₽
Стоимость потери посетителя:              0 ₽ (он просто закроет вкладку)
```

Размытые фото, путаный каталог, непонятная навигация — и человек ушёл. Навсегда. К конкуренту, у которого кнопка «Купить» на видном месте.

### 3. Тщеславные метрики

```
❌ «У нас выручка 2 миллиона в месяц!»
✅ «У нас прибыль 200 000 в месяц при ROAS 4.5»

Выручка без прибыли = благотворительность
```

### 4. Нет повторных продаж

Привлечь нового клиента стоит в 5–7 раз дороже, чем удержать старого. Но 90% магазинов вкладывают весь бюджет в привлечение и забивают на тех, кто уже купил.

## Юнит-экономика

Считайте до запуска, а не после. Формула простая:

```
Прибыль с заказа = Средний чек
                  - Себестоимость товара
                  - Стоимость привлечения клиента (CAC)
                  - Доставка
                  - Эквайринг (~2%)
                  - Возвраты (закладывайте 5–15%)
                  - Упаковка
                  - Операционные расходы (на 1 заказ)
```

### Пример: магазин чехлов для телефонов

```
Средний чек:                    1 500 ₽
Себестоимость:                  - 400 ₽
CAC (реклама):                  - 450 ₽
Доставка:                       - 200 ₽
Эквайринг (2%):                 - 30 ₽
Возвраты (10%):                 - 150 ₽
Упаковка:                       - 50 ₽
Операционка:                    - 100 ₽
─────────────────────────────────────────
Прибыль с заказа:               120 ₽ (8%)
```

> ⚠️ При марже 8% любое повышение стоимости рекламы или увеличение возвратов делает бизнес убыточным. Это зомби-экономика.

### Сравнение: плохая vs хорошая юнит-экономика

```
                        Чехлы (плохо)    Кожаные изделия (хорошо)
───────────────────────────────────────────────────────────────────
Средний чек              1 500 ₽          8 000 ₽
Себестоимость             400 ₽           2 000 ₽
CAC                       450 ₽            800 ₽
Маржа                      8%              45%
Повторные покупки         Редко            Часто
Конкуренция               Ад               Умеренная
Итог                     Зомби            Живой бизнес
```

## Повторные продажи

Это то, что отличает живых от зомби. Инструменты:

### Email-рассылка

```
Стоимость:     ~0 ₽ за письмо (при базе до 1000)
Конверсия:     3–8% (vs 1–2% у платной рекламы)
ROI:           до 4200% (самый высокий из всех каналов)
```

Не спам «купи-купи-купи». А полезный контент + персональные предложения.

### Telegram-канал

```
Стоимость подписчика:    15–50 ₽
Open rate:               60–80% (vs 20% у email)
Охват поста:             30–50% базы
```

Свой канал — это аудитория, которая не зависит от алгоритмов рекламных площадок.

### Программа лояльности

```
Простая схема:
- 1 покупка    →  нет бонусов
- 2 покупка    →  5% скидка
- 3+ покупка   →  10% скидка + бесплатная доставка

Результат: LTV клиента вырастает в 2–3 раза
```

### Формула жизнеспособности

```
Если CAC > LTV   →  бизнес мёртв
Если CAC < LTV   →  бизнес жив
Если CAC < LTV/3 →  бизнес масштабируется

CAC = стоимость привлечения клиента
LTV = сколько клиент принесёт за всё время
```

## Ниша vs «всё для всех»

```
Магазин «аксессуары и подарки»:

  Конкуренция:          ★★★★★ (тысячи магазинов)
  Экспертиза:           Никакая
  Лояльность:           Нулевая
  Стоимость рекламы:    Высокая
  Средний чек:          Низкий


Магазин кожаных обложек ручной работы:

  Конкуренция:          ★★☆☆☆
  Экспертиза:           Глубокая
  Лояльность:           Высокая
  Стоимость рекламы:    Низкая
  Средний чек:          Высокий
```

> Лучше быть большой рыбой в маленьком пруду, чем маленькой рыбой в океане.

Примеры удачных ниш:

| Ниша | Средний чек | Конкуренция | Повторные покупки |
|---|---|---|---|
| Специализированный чай | 1 500–3 000 ₽ | Низкая | Высокие (подписки) |
| Минеральная косметика | 2 000–5 000 ₽ | Средняя | Высокие |
| Аксессуары для винила | 3 000–15 000 ₽ | Низкая | Средние |
| Товары для левшей | 500–3 000 ₽ | Очень низкая | Средние |
| Эргономичная мебель для WFH | 15 000–80 000 ₽ | Средняя | Низкие, но чек огромный |

## Платформы для запуска

| Платформа | Стоимость | Сложность | Для кого |
|---|---|---|---|
| **Tilda** | от 750 ₽/мес | Низкая | MVP, тест ниши, до 100 товаров |
| **Shopify** | от $39/мес | Низкая | Серьёзный старт, глобальный рынок |
| **WooCommerce** | Бесплатно (+ хостинг) | Средняя | Контроль, гибкость, WordPress |
| **InSales** | от 990 ₽/мес | Низкая | Рунет, интеграции с СДЕКом, Почтой |
| **1C-Битрикс** | от 35 900 ₽ | Высокая | Крупный бизнес, сложные каталоги |
| **Кастом (Next.js)** | $5 000+ | Высокая | Уникальные требования |

### Как выбрать

```
Тестируешь нишу?          → Tilda / InSales
Запускаешь серьёзно?      → Shopify / WooCommerce
Нужна кастомизация?       → WooCommerce / Next.js
Крупный каталог?          → 1C-Битрикс
Глобальный рынок?         → Shopify
```

## Чек-лист перед запуском

```markdown
Юнит-экономика:
- [ ] Посчитана маржинальность каждого товара
- [ ] CAC < LTV/3
- [ ] Заложены возвраты (5–15%)
- [ ] Операционные расходы учтены

Товар:
- [ ] Есть ответ на «почему у нас, а не на Ozon»
- [ ] Ниша конкретная, не «всё для всех»
- [ ] Фото — свои, качественные, минимум 5 на товар
- [ ] Описания отвечают на вопросы покупателя

Сайт:
- [ ] Загрузка < 3 секунд
- [ ] Мобильная версия работает идеально
- [ ] Путь до покупки — максимум 3 клика
- [ ] Оплата: карты + СБП минимум

Маркетинг:
- [ ] Настроена аналитика (Метрика + GA4)
- [ ] Есть стратегия привлечения (не только реклама)
- [ ] Подготовлен email-сбор и рассылка
- [ ] Есть Telegram-канал или план на него

Повторные продажи:
- [ ] Есть механизм сбора контактов
- [ ] Есть план email/Telegram коммуникации
- [ ] Продумана программа лояльности
```

## Заключение

Интернет-магазин — это не сайт с каталогом. Это продукт. И как любой продукт, он либо решает конкретную проблему конкретного человека лучше, чем альтернативы — либо он никому не нужен.

Пять правил для живого магазина:

1. **Считай цифры** — юнит-экономика до запуска, не после
2. **Суживай нишу** — чем уже, тем лояльнее аудитория
3. **Продавай повторно** — новый клиент дорого, старый — бесплатно
4. **Делай качественно** — фото, описания, сервис на уровне
5. **Отвечай на вопрос** — почему здесь, а не на маркетплейсе

Если вы не можете объяснить в одном предложении зачем покупать именно у вас — покупать у вас не будут.
